🎬 JW 影片連結提取器 (JW Video Link Extractor)

這是一個純前端的網頁實用工具，旨在幫助使用者快速從 JW.org 的「分類頁面（Categories）」中，自動遞歸提取所有影片連結，並提供強大且便利的「區間勾選」與「一鍵批量複製」功能。

✨ 核心特色功能

🚀 深度遞歸提取：自動穿透並掃描分類下的所有子目錄，最高可一次提取 300 筆影片連結。

🌍 智慧語言解析：自動辨識網址中的語言代碼（如 cmn-hant），並透過內建對應表或線上 API 動態轉換為 JW 系統的內部語言代碼（如 CH）。

✅ 強大的選取面板：

單選 / 全選：直覺的打勾介面。

區間選取：支援輸入起始與結束號碼（例如：11 ~ 20），精準批次選取所需的影片。

📋 一鍵複製帶走：將所有打勾的影片，自動轉化為完整網頁連結，一鍵複製到剪貼簿。

📖 內建防呆教學：內建精美的圖文彈出視窗（Modal）說明，首次開啟自動提示。

🛠️ 即時除錯日誌：內建 Debug Log 面板，運作狀態、API 請求結果一目了然。

步驟 1：取得分類網址

前往 JW.org 的影片庫，進入您想要抓取的「分類頁面」。

步驟 2：貼上並提取

將複製下來的網址貼入本工具上方的輸入框中，並點擊 「立即提取影片」 按鈕。
系統會在背景自動連線至 JW API 進行資料抓取，您可以透過下方的「系統執行日誌」觀察進度。

⚠️ 注意事項與限制

不支援「搜尋結果」頁面：
如果您的網址包含 /search/（例如透過右上角放大鏡搜尋出來的頁面），本工具 無法 提取。因為搜尋結果沒有固定的 Category ID。此情況建議使用瀏覽器的開發者工具 (F12) 透過腳本提取。

提取數量上限：
為避免對官方伺服器造成過大負擔並確保瀏覽器效能，目前單次提取上限設定為 300 筆影片。

API 跨域限制 (CORS)：
本工具使用官方公開的 b.jw-cdn.org API，由於該 API 允許跨域請求，因此本工具可直接在客戶端瀏覽器運行。若未來官方更改 CORS 政策，可能需要透過 Proxy 伺服器才能運作。
